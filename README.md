# 3d-forge
Read/Write quantized-mesh tiles

## Getting started

    make all

## Interactive programming

    source venv/bin/activate
    ipython
    run {yourScript}.py

#### Enter debug mode

    %debug

#### Get the latest traceback

    %tb


## Styling

#### Check styling

    make lint

#### Fix styling (only pep8 errors)

    make autolint
